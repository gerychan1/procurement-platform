# Procurement Platform

A comprehensive procurement platform built with Next.js and React that allows users to:
- Browse and search for items
- Add items to cart
- Submit purchase requests
- Manage categories
- Track orders

## Technologies Used
- Next.js 13+
- React
- TypeScript
- Tailwind CSS
- Shadcn UI Components
